# 게시판 만들어보기

## 기술스택
- spring boot 3.5.13 
4.0.5는 최신이라 레퍼런스가 없다고 한다.
- postgresql
데이터베이스 수업에서 postgresql 사용해서, 사용하고 싶었다. 게시판은 관계형데이터베이스에 적합. 근데 mysql이든 postgresql이든 상관없다고 생각했음.

## 목표 기능 명세
- 게시글 작성
- 게시글 조회
- 게시글 수정
- 게시글 삭제
- 게시글 목록 조회 (페이징)

## 시간이 남았을 때 추가
- 검색(제목, 작성자, 내용)
- 인증(회원가입, 로그인)
- 인가(권한설정)
- 조회수
- 댓글
- 파일업로드
- 배포
- 기타 등등
